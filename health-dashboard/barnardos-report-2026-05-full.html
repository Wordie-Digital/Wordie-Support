Recommended actions

Prioritised list of actions for this month

HIGH PRIORITY
Defer Convert Experiments analytics script and lazy-load FontAwesome icons
These two changes account for ~1,560ms of blocking time. Expected improvement: +15–20 PageSpeed points. Google uses mobile speed as a ranking factor — this directly impacts organic search traffic to Barnardos' donation and service pages.
HIGH PRIORITY
Audit and reduce render-blocking CSS — Bootstrap and FontAwesome total 47KB of blocking CSS
Reducing render-blocking CSS improves Speed Index and LCP, making the page feel significantly faster. Replace with utility-only classes where possible.
MEDIUM PRIORITY
Enable WP Engine CDN caching and configure static asset cache headers
Low-effort infrastructure change with meaningful impact on repeat visitor load times. Can be done without code changes.
MEDIUM PRIORITY
Remove jQuery Migrate (5.2KB) — no legacy jQuery code should require it on a current build
Reduces page weight and eliminates an unnecessary legacy dependency from every page load.
